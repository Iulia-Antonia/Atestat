2.PARITATEA


Funcțiile pare și impare


Definiția : Mulțime simetrică
O mulțime D, inclusa in R se numește mulțime simetrică dacă, luând un element x din D , atunci și -x este din D.


Definiția : Funcția pară
Fie D o mulțime nevidă și funcția f:D->R.
Spunem că f este o funcție pară, dacă f(-x)=f(x) oricare ar fi x din D.


Definiția: Funcția impară
Funcția f:D->R, cu D mulțime simetrică, se numește impară, dacă f(-x)=-f(x) oricare ar fi x din D.
