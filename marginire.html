1.Mărginirea

Funcția f:A->B se numește mărginită, dacă există a,b din R , astfel încât a<=f(x)<=b.
Cu alte cuvinte, mulțimea valorilor funcției f este mărginită.
Observație:
Funcția f este mărginită dacă și numai dacă există a,b din R astfel încât f(x) aparține intervalului [a,b], oricare ar fi x din mulțimea A, ceea ce este echivalent cu a spune că funcția f este mărginită dacă și numai dacă există a,b din R, astfel încât f(A) să aparțină intervalului [a,b].
Exemple:
1. Funcția f:R->R, f(x)=sin x este mărginită, deoarece sin x aparține intervalului [-1,1], oricare ar fi x din R.
2. Funcția f:[-2,3]->R, f(x)=-2x+1 este mărginită, deoarece f(x) aparține intervalului [-5,5] oricare ar fi x din [-2,3].
