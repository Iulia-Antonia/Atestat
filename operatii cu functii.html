5.Operații cu funcții


Definitie(Operatii cu functii):



ADUNAREA


Functia (f+g): R definita prin (f+g) (x) = f(x)+g(x), f din A. - se numește suma dintre funcția f și funcția g.

ÎNMULȚIREA


Functia (f*g) A R definita prin (f*g ) (x) = f (x) * g(x), se numește produsul dintre funcția f și funcția g.

ÎMPĂRȚIREA


Functia (f/g A - R definita prin ( f/g ) (x) = f(x)/f(g), se numește câtul dintre funcția f și funcția g.

DIFERENȚA


Functia (f-g): R definita prin (f-g) (x) = f(x)-g(x) se numeșe diferența dintre funcția f și funcția g.

Exemplu:
Fie f, g : R R, f(x) = 3x+1, g(x) = - x +3.
Atunci f + g, f- g, f*g : R R sunt definite astfel:
(f + g )(x) = f(x) + g(x) = 3x + 1 - x +3 = 2x + 4.
(f - g)(x) = f(x) - g(x) = 3x+1 -x + 3 = 4x - 2.
(f*g)(x) = f(x)*g(x) = (3x + 1)(-x + 1) = -3x2+2x+1.



Proprietăți ale adunării funcțiilor



Fie F (A, R) multimea tuturor functiilor definite pe A cu valori in R. Atunci are loc:
Teorema: Pentru operatia de adunare pe F (A, R) au loc proprietatile:
• (f +g) + h = f + (g + h), (adunarea functiilor este asociativa);
•f + g = g + f (adunarea functiilor este comutativa);
•exista functia 0 f(x)+0(x)=f(x) oticare ar fi x din domeniu (0 se numeste functie nula si este element neutru pentru adunarea functiilor);
•Există -f astfel incat f-f=-f+f = 0( orice functie f are o opusa –f)



Proprietăți ale înmulțirii funcțiilor



Fie F (A, R) multimea tuturor functiilor definite pe A cu valori in R. Atunci are loc:
Teorema: Pentru operatia de inmultire pe F (A, R), au loc proprietatile:
• (f *g) * h = f * (g * h), (înmulțirea funcțiilor este asociativă);
•f * g = g * f (adunarea functiilor este comutativa);
•exista functia 1 f(x)+1(x)=f(x) oticare ar fi x din domeniu (1 se numeste functie nula si este element neutru pentru adunarea functiilor);
•Există f' astfel incat f*(f')=(f')*f = 1( orice functie f are o opusa f')



Propozitie: Inmultirea este distributiva in raport cu adunarea pe F(A, R), adica:
f * (g + h) =fg + f h, oricare ar fi f, g, h din F (A, R).
